# Documentation

This directory contains all documentation for the Viscum project.

## Contents

### Documentation Files

- **[ALGORITHM.md](ALGORITHM.md)** - Detailed algorithm explanation with mathematical formulas
- **[FLOWCHART.md](FLOWCHART.md)** - Mermaid flowcharts (detailed and simplified versions)
- **[TUTORIAL_SCRIPT.md](TUTORIAL_SCRIPT.md)** - Video tutorial recording guide

### Flowchart Images

Add your exported flowchart PNG files here:

- `flowchart-detailed.png` - Full algorithm flowchart (for technical documentation)
- `flowchart-simplified.png` - 7-step overview (for presentations)

**How to generate:**
1. Open [FLOWCHART.md](FLOWCHART.md)
2. Copy the Mermaid code
3. Go to https://mermaid.live
4. Paste and export as PNG
5. Save here with the names above
