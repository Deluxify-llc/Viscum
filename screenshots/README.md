# Screenshots

This directory is for screenshots and visual documentation of the Viscum tool.

## Suggested Screenshots

1. **gui-main.png** - Main GUI interface showing all input fields
2. **roi-selection.png** - ROI selection process on video preview
3. **tracking-results.png** - Example tracking visualization with ball detection
4. **results-window.png** - Results popup showing viscosity calculation
5. **confidence-plot.png** - Example confidence plot over time
6. **velocity-plot.png** - Example velocity vs time graph
7. **diameter-plot.png** - Example ball diameter measurements

## How to Add Screenshots

1. Run the tool with `examples/mineral_oil.mp4`
2. Take screenshots at each step
3. Save them in this directory with descriptive names
4. Reference them in the main README.md

## File Formats

- Use **PNG** format for crisp UI screenshots
- Keep file sizes reasonable (< 500 KB per image)
- Use descriptive filenames (lowercase with hyphens)
